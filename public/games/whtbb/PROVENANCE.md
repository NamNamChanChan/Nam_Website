The preservation source describes this as an offline-fixed build. Features that
depended on the retired Facebook service, including statistics, challenges and
leaderboards, are not expected to work.

The item credits PandaFake⚡ with archiving the game and modifying it to run
offline. PandaFake⚡ thanks BattleAncient (Rachid) for supplying the main file
and floydian (Alejandro) for helping to fix glitches. The item was uploaded to
Internet Archive by PandaFake⚡ on 2025-05-29.

An update recorded on the item as `9/8/25` lists these changes:

- Fixed the invisible correct-sign glitch in Car Path.
- Added a Hexagon Path reset sequence based on the iPhone version.
- Removed the obsolete landscape-only score board.
- Removed the upload-score pop-up.

The website runs this file with a repository-owned copy of Ruffle 0.3.0. Neither
the game file nor its emulator is fetched from an external service at runtime.
Credit for modern browser emulation belongs to the Ruffle contributors.

`preview.png` was captured from this local preservation copy's main menu and is
used as the player's click-to-play preview image.
