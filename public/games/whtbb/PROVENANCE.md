The preservation source describes this as an offline-fixed build. Features that
depended on the retired Facebook service, including statistics, challenges and
leaderboards, are not expected to work.

The website runs this file with a repository-owned copy of Ruffle 0.3.0. Neither
the game file nor its emulator is fetched from an external service at runtime.

`preview.png` was captured from this local preservation copy's main menu and is
used as the player's click-to-play preview image.
